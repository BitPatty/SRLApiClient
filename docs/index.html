SRLApiClient

A .NET Standard 2.0 wrapper around the SpeedRunsLive API.

Quick Start Notes:

Installation

You can install the client via NuGet:

#.Net
dotnet add package SRLApiClient

#Package Manager
Install-Package SRLApiClient

Requirements

The SRL API Client supports any platform that implements .NET Standard 2.0.

Dependencies

none
